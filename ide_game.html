Deskripsi Dino Run

Dino Run adalah sebuah game offline dengan berkarakter T-Rex. Game ini tidak membutuhkan sebuah internet. Game ini diciptakan untuk kalangan semua umur, karena mudah untuk dipahami dari dalam segi permainannya.

Konsep Dino Run

Konsep yang akan kita buat dalam game ini adalah kita hanya perlu melewati rintangan yang menghadang perjalanan Dino. Game ini mempunyai latar tempat yang bertema hutan. Tujuan akhir dari game ini adalah mencari score sebanyak-banyaknya sampai kalah karena menabrak batu.

Alur Dino Run

Dino Run adalah game dengan tema melarikan diri, yang dikembangkan oleh Kelompok 13. Awal mula game ini ada sebuah Dinosaurus yang melarikan diri mengalami kepunahan karena dipicu oleh hantaman asteroid terhadap Bumi. Peristiwa tersebut terjadi pada 66 juta tahun yang lalu. Bahkan, hantaman asteroid yang dahsyat telah menyebabkan musim dingin ekstrem kala itu dan hilangnya sekitar 75 persen kehidupan di Bumi. Alur game dalam Dino Run Permainan ini mengharuskan para pemainnya untuk terus berlari melewati semua halangan yang ada, melompati rintangan yang ada. Kontrolnya pun tidaklah sulit, cukup menggunakan spasi untuk melompat dan menghindar. Tujuan diciptakannya permainan ini adalah untuk agar dinosaurus bisa bertahan hidup selama mungkin dalam sekali bermain. Namun jika dinosaurus tertabrak rintangan maka akan kalah dan game akan mengulang ke titik awal lagi.

Cara Bermain Dino Run

1. Otomatis memulai permainan dengan berlari

2. Score tercatat saat dino mulai berlari hingga game over

3. Dino melompati/menghindari batu dengan cara menekan spasi

4. Kalah jika tertabrak batu

5. Merefresh website jika ingin bermain lagi
